

# הפיכת AquaFlow לאפליקציה נייטיבית (App Store / Google Play)

## מה זה אומר?

האפליקציה שלך כבר בנויה כ-web app מותאם למובייל. כדי להפוך אותה לאפליקציה אמיתית שאפשר להוריד מחנויות האפליקציות, נשתמש בטכנולוגיה בשם **Capacitor** — שעוטפת את האפליקציה הקיימת באריזה נייטיבית בלי לשנות את הקוד.

## מה צריך לעשות

### שלב 1 — הגדרת Capacitor בפרויקט
- התקנת חבילות Capacitor (core, cli, ios, android)
- יצירת קובץ הגדרות (`capacitor.config.ts`) עם שם האפליקציה והמזהה שלה
- הגדרת חיבור לסביבת הפיתוח לצורך בדיקות חיות

### שלב 2 — עבודה במחשב שלך
אחרי שנגדיר את Capacitor בפרויקט, תצטרך לבצע את השלבים הבאים במחשב שלך:

1. **ייצוא לגיטהאב** — דרך כפתור "Export to GitHub" ב-Lovable
2. `git clone` + `npm install`
3. `npx cap add ios` ו/או `npx cap add android`
4. `npx cap update ios` / `npx cap update android`
5. `npm run build`
6. `npx cap sync`
7. `npx cap run android` או `npx cap run ios`

### דרישות חומרה
- **אפל (iOS)**: חייב Mac עם Xcode מותקן + חשבון Apple Developer ($99/שנה)
- **אנדרואיד**: צריך Android Studio (עובד על Windows/Mac/Linux) + חשבון Google Play ($25 חד-פעמי)

## מה לא ישתנה
- כל הלוגיקה, העיצוב, התשלומים, ניהול בתי ספר — הכל נשאר בדיוק כמו שהוא
- האפליקציה הנייטיבית מריצה את אותו קוד בדיוק

## חלופה פשוטה יותר — PWA
האפליקציה שלך כבר מוגדרת כ-PWA (יש manifest.json ואייקונים). משתמשים יכולים כבר עכשיו להתקין אותה מהדפדפן ישירות למסך הבית — בלי חנות אפליקציות. זה עובד על אנדרואיד ואייפון.
